Exemplo

Veja o código de exemplo para utilizar o botão do Pra Construir:

    document.addEventListener("DOMContentLoaded", function () {
        let praconstruirBtn = new PraconstruirBtn('#praconstruirBtn');
        let listOfProducts = [[111007,9531,10],[111006,9531,30]];
        praconstruirBtn.addAllProducts(listOfProducts);
    });

Veja que o botão foi instanciado após o carregamento do DOM, e foi utilizada a função addAllProducts para inserir uma array contendo os produtos, unidades e quantidades. Ao clicar no botão, esta lista de id's será formatada e será aberta uma janela do Pra Construir, onde será carregada a lista de produtos pronta para solicitar orçamento.

Para buscar as id's de produtos e unidades pra utilização em seu site, utilize o link https://praconstruir.com.br/developer/painel/busca
